Whether or not the streaming API should be used to receive flag updates.

UseLdd
True if this client should use the LaunchDarkly relay in daemon mode, instead of subscribing to the streaming or polling API.
UserAgentType
A string that will be sent to LaunchDarkly to identify the SDK type.

The number of user keys that the event processor can remember at any one time, so that duplicate user details will not be sent in analytics events.
UserKeysFlushInterval
The interval at which the event processor will reset its set of known user keys. The default value is five minutes.
