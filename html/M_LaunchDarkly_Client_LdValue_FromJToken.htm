Remarks

The Newtonsoft.Json type JToken is being phased out of the SDK because it is mutable (so it is possible to unintentionally modify a JSON object or array that is being used elsewhere), and in order to reduce third-party dependencies. However, since parts of the current API still use this type, FromJToken(JToken) and AsJToken provide a simple conversion.

In order to avoid the mutability problem, this method performs a deep copy of any JObject or JArray value.

FromJToken(JToken) and AsJToken are immediately deprecated, to encourage developers to migrate away from using JToken-based SDK methods as soon as possible.
