Remarks

This method creates a "custom" analytics event containing the specified event name (key), user properties, and optional custom data. If you do not need custom data, pass Null for the last parameter or simply omit the parameter.

Note that event delivery is asynchronous, so the event may not actually be sent until later; see Flush.
