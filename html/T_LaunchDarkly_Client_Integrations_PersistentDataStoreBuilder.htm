Remarks

For a persistent data store (e.g. a database integration), the store implementation will provide an IPersistentDataStoreFactory or IPersistentDataStoreAsyncFactory that implements the specific data store behavior. The SDK then provides additional options for caching; those are defined by this type, which is returned by PersistentDataStore(IPersistentDataStoreFactory). Example usage:

Copy
var myStore = Components.PersistentDataStore(Redis.DataStore())
    .CacheSeconds(45);
var config = Configuration.Builder(sdkKey)
    .DataStore(myStore)
    .Build();
