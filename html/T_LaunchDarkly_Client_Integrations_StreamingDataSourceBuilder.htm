Remarks

By default, the SDK uses a streaming connection to receive feature flag data from LaunchDarkly. If you want to customize the behavior of the connection, create a builder with StreamingDataSource, change its properties with the methods of this class, and pass it to DataSource(IUpdateProcessorFactory).

Setting Offline(Boolean) to will supersede this setting and completely disable network requests.

Examples
Copy
var config = Configuration.Builder(sdkKey)
    .DataSource(Components.PollingDataSource()
        .PollInterval(TimeSpan.FromSeconds(45)))
    .Build();
