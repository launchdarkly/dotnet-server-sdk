Remarks
Obtain an instance of this class by calling Builder(String). All of the builder methods for setting a user attribute return a reference to the same builder, so they can be chained together (see example). Some of them have the return type IUserBuilderCanMakeAttributePrivate rather than IUserBuilder; those are the user attributes that can be designated as private.
Examples
Copy
var user = User.Builder("my-key")
    .Name("Bob")
    .Email("test@example.com")
    .Build();
