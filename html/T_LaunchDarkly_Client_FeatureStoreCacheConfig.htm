Remarks

The built-in InMemoryFeatureStore does not use this class; it is meant for database implementations.

This is an immutable class that uses a fluent interface. Obtain an instance by getting the static value Disabled or Enabled; then if desired, you can use chained methods to set other properties:

Copy
FeatureStoreCacheConfig.Enabled.WithTtlSeconds(30);
