Remarks
As of this version’s release date, the LaunchDarkly service does not support the metricValue parameter. As a result, calling this overload of Track will not yet produce any different behavior from calling Track(String, JToken, User) without a metricValue. Refer to the SDK reference guide for the latest status: https://docs.launchdarkly.com/docs/dotnet-sdk-reference#section-track
