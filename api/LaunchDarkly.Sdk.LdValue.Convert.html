Field Value
Type Description
LdValue.Converter<System.Int64>
Remarks

Its behavior is consistent with Of(Int64) and AsLong.

Note that the LaunchDarkly service, and most of the SDKs, represent numeric values internally in 64-bit floating-point, which has slightly less precision than a signed 64-bit langword_csharp_long; therefore, the full range of langword_csharp_long values cannot be accurately represented. If you need to set a user attribute to a numeric value with more significant digits than will fit in a double, it is best to encode it as a string.
