Remarks

This is the default for all new flags created with Flag(String). The flag will have two variations, true and false (in that order); it will return false whenever targeting is off, and true when targeting is on if no other settings specify otherwise.

Type Description
TestData.FlagRuleBuilder

a TestData.FlagRuleBuilder; call ThenReturn(Boolean) or ThenReturn(Int32) to finish the rule, or add more tests with another method like AndMatch(UserAttribute, LdValue[])
Remarks

For example, this creates a rule that returns true if the name is "Patsy" or "Edina": 

    testData.Update(testData.Flag("flag-key")
        .IfMatch(UserAttribute.Name, LdValue.Of("Patsy"), LdValue.Of("Edina"))
        .ThenReturn(true));
