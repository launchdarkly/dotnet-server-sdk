Class LdValue.Converter<T>

Defines a conversion between LdValue and some other type.

Type Parameters
Name Description
T

the type to convert from/to
Remarks

Besides converting individual values, LdValue.Converter<T> provides factory methods like ArrayOf(T[]) which transform a collection of the specified type to the corresponding LdValue complex type.

There are type-specific instances of this class for commonly used types in LdValue.Convert, but you can also implement your own.
