Namespace LaunchDarkly.Sdk.Server.Integrations

Tools for configuring how the SDK connects to LaunchDarkly, or to other software components.

This package contains the configuration builders for standard SDK components such as StreamingDataSourceBuilder; the PersistentDataStoreBuilder builder for use with database integrations (the specific database integrations themselves are provided by add-on libraries); and the test fixtures FileData and TestData.
