Remarks

In the LaunchDarkly model, a flag can have any number of rules, and a rule can have any number of clauses. A clause is an individual test such as "name is 'X'". A rule matches a context if all of the rule's clauses match the context.

To start defining a rule, use one of the flag builder's matching methods such as IfMatchContext(ContextKind, String, LdValue[]). This defines the first clause for the rule. Optionally, you may add more clauses with the rule builder's methods such as AndMatchContext(ContextKind, String, LdValue[]). Finally, call ThenReturn(Boolean) or ThenReturn(Int32) to finish defining the rule.

Remarks

For example, this creates a rule that returns true if the name is "Patsy" and the country is "gb": 

    testData.Update(testData.Flag("flag-key")
        .IfMatch("name", LdValue.Of("Patsy"))
        .AndMatch("country", LdValue.Of("gb"))
        .ThenReturn(true));
